# 第 2 课：进阶用法

## 表格

| 字段 | 类型 | 说明 |
|------|------|------|
| `name` | string | 名称 |
| `age` | number | 年龄 |

## 折叠面板

import { Steps } from '@astrojs/starlight/components';

<Steps>
  1. **第一步**

     做点什么。

  2. **第二步**

     再做点什么。

  3. **第三步**

     完成。
</Steps>

> ⚠️ `<Steps>` 里必须用 `1. 2. 3.` 这种有序列表，每项内部缩进 4 空格。
> 不要在里面用 `###` 标题。

## 插入图片

把图片放进 `src/assets/`，然后引用：

```mdx
import myImage from '../../assets/screenshot.png';

![截图说明](myImage)
```

> ⚠️ 引用图片必须用 `.mdx` 文件（`.md` 不支持 `import`）。
