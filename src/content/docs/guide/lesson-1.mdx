## 正文

> 💡 提示框：Starlight 支持 GitHub 风格的 aside 语法，自动渲染成带图标的彩色框。

> [!NOTE]
> 这是一条 note（蓝色，默认）。

> [!TIP]
> 这是一条 tip（绿色）。

> [!WARNING]
> 这是一条 warning（黄色）。

> [!CAUTION]
> 这是一条 caution（红色，表示危险操作）。

也可以用组件版 `<Aside>`：

import { Aside } from '@astrojs/starlight/components';

<Aside type="note" title="组件版">
  效果和上面一样，但用 JSX 写法。
</Aside>
