## 关于这个站

这里记录了利用 **AI 辅助**完成的几个典型课程设计项目 —— 从 Java 桌面游戏、跨平台代码学习工具，到这个展示站本身，再到自动生成报告格式的 Skill 工具。

每个项目均附有完整的**重构计划、大模型提示词与详细工作日志**，实现了软件生命周期中从需求到发布的精细复盘与留档。

> 站点使用 [Astro](https://astro.build) + [Starlight](https://starlight.astro.build) 构建，托管于 [Cloudflare Pages](https://pages.cloudflare.com)。
