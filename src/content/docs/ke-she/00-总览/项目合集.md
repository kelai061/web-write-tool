### 1. [基础课设 · GenshinRPG](file:///ke-she/01-基础课设-genshinrpg/基础课设项目/)
- **项目概述**：基于 Java Swing 和 MySQL 数据库构建的具有原神风格的文字角色扮演游戏（RPG）。
- **主要内容**：
  - 精简直白的界面交互设计与完整的数据库操作。
  - 基于 AI 自动生成的两阶段重构计划，优化包结构、整理通用工具类并为关键链路添加全量中文注释。
  - 内置精简数据库的独立 ZIP 分发包打包实践与运行脚本。
- **文档入口**：[基础课设项目说明](file:///ke-she/01-基础课设-genshinrpg/基础课设项目/)

### 2. [代码学习工具 · Supplement](file:///ke-she/02-代码学习工具-supplement/代码学习工具项目/)
- **项目概述**：采用逆向拆解学习法理念设计的代码刷题与辅助工具，支持多平台和移动端适配。
- **主要内容**：
  - 多分辨率适配、系统状态栏遮挡等常见移动端 UI 兼容性问题的全面修复方案。
  - Gradle 多模块构建配置的自动获取与同步指南。
  - 三轮 Bug 修复与迭代的完整工作日志及验证报告。
- **文档入口**：[代码学习工具项目说明](file:///ke-she/02-代码学习工具-supplement/代码学习工具项目/)

### 3. [课设展示网站](file:///ke-she/03-课设展示网站/课设展示网站项目/)
- **项目概述**：集中托管和检索所有课程设计笔记与成果的扁平化文档网站。
- **主要内容**：
  - 基于 Astro 5 静态站点引擎与 Starlight 文档模版进行的主题颜色定制与布局优化。
  - Cloudflare Pages 自动化构建和持续部署（CI/CD）方案。
  - 本地与线上环境 404 路径大小写敏感性兼容修复调研。
- **文档入口**：[课设展示网站项目说明](file:///ke-she/03-课设展示网站/课设展示网站项目/)

### 4. [格式 Skill 制作](file:///ke-she/04-格式skill制作/格式skill制作/)
- **项目概述**：基于大语言模型开发的一键生成学术/工程规范 Word 报告的 AI 提效助手。
- **主要内容**：
  - 基于 Node.js 与 Puppeteer 实现的 Carbon 窗口渲染与截图自动化工具，自动添加指定文字水印。
  - 基于 Python `python-docx` 库和 Word 模板进行的一键文本自动排版、表格绘制与目录生成。
  - 自动化脚本的验证报告与 HANDOFF 工作交接手册。
- **文档入口**：[格式 Skill 制作说明](file:///ke-she/04-格式skill制作/格式skill制作/)
