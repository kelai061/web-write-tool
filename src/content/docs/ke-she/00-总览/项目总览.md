## 1. 基础课设 · GenshinRPG

### 简介
一个基于 Java Swing + MySQL 架构的角色扮演游戏（RPG）桌面程序。项目实现了完整的角色属性管理、动态战斗系统和基于本地数据文件的自动进度保存功能，结构清晰，适合 Java 基础教学与工程实践复盘。

### 技术栈
- **语言**: Java (SE)
- **GUI 框架**: Java Swing (AWT / CardLayout)
- **数据库**: MySQL, JDBC
- **协作开发**: AI 智能体辅助重构与代码审查

### 最终成果
- **代码重构与链路注释**：在不破坏原有业务逻辑的前提下，实现了代码结构的大规模模块化拆分。大幅降低函数嵌套层级，并在核心交互按钮处增加了直观的事件调用链路注释，极大地提升了初学者的可读性。
- **环境隔离与一键分发**：成功编译了内置微型数据库的独立运行包（Zip 格式），解决了跨设备运行时复杂的 JDK 环境与 MySQL 驱动配置报错，实现了“解压即运行”的绿色分发。

## 2. 代码学习工具 · Supplement

### 简介
一款基于逆向拆解学习理念的代码刷题与算法辅助工具。项目专门针对移动端（手机与平板）的各种尺寸屏幕进行了自适应排版优化，能够动态同步开发配置，是探索移动端兼容性适配的典型案例。

### 技术栈
- **开发环境**: Android Studio, Android SDK
- **构建工具**: Gradle
- **语言/库**: Java/Kotlin, SQLite

### 最终成果
- **UI 兼容性适配**：成功攻克了 Android 状态栏遮挡应用顶部的经典排版问题，针对平板与手机多分辨率进行了适配，提供了无缝的全面屏交互体验。
- **Gradle 依赖与后端同步**：完成了 Gradle 多版本环境下的依赖项自动获取与本地同步构建，修复了因后端配置缺失导致的 MSI 安装包连接错误。
- **构建分发包优化**：相较于不稳定的 msi 安装器，转而采用纯源码一键重新构建生成 Zip 文件的方式，将分发成功率提升了 95% 以上。

## 3. 课设展示网站

### 简介
用于集中展示、在线托管与版本归档所有课程设计作品及技术日志的现代化文档展示站点（即本站）。

### 技术栈
- **核心框架**: Astro 5 (Island Architecture 静态引擎)
- **页面主题**: @astrojs/starlight (响应式文档主题)
- **搜索服务**: Pagefind (静态本地全文检索)
- **托管平台**: Cloudflare Pages (边缘分发 CI/CD)

### 最终成果
- **内容自动化渲染**：建立了一套从本地 Markdown 文档向现代化静态 HTML 网页自动转换与编译的排版机制，保证了多级标题与相对路径资源在编译后的正确解析。
- **CI/CD 自动化部署**：配置了 Git 托管与 Cloudflare Pages 构建流水线，实现代码提交后自动触发全球边缘节点秒级打包与热重载发布，绑定了自定义域名。

## 4. 格式 Skill 制作

### 简介
基于大语言模型（LLM）定制开发的格式化排版 AI 助手（Skill）。该工具支持输入原始粗糙文本和代码文件，自动生成符合学术与工程排版规范的 Microsoft Word (.docx) 格式课程设计报告。

### 技术栈
- **核心逻辑**: Python (python-docx)
- **截图渲染**: Node.js, Puppeteer, Carbon API
- **自动化流**: Shell / PowerShell 自动化脚本

### 最终成果
- **Carbon 风格本地渲染服务**：将网页版 Carbon 渲染器在本地 localhost:3000 进行离线托管，结合 Puppeteer 无头浏览器实现了 100% 还原的 Mac 风格代码高亮截图，解决了大片代码段在 Word 中排版杂乱的问题。
- **Word 报告一键装配**：完成了多篇真实学生课设报告的批量自动化格式排版测试，自动提取封面字段、智能过滤冗余代码并转换插图占位符，生成可直接打印 of Docx 文档。
