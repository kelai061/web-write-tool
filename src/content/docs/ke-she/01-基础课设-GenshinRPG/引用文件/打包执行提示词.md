### DBConnector.java 改动（精确）

文件：`src\com\genshin\rpg\dao\DBConnector.java`，定位到含 `E:\\SQL\\mysql` 的那段（`connectDB` 方法内，约 50-65 行）。
把：

```java
File mysqlExe = new File("E:\\SQL\\mysql\\bin\\mysqld.exe");
File myIni = new File("E:\\SQL\\mysql\\my.ini");
if (mysqlExe.exists() && myIni.exists()) {
    ProcessBuilder pb = new ProcessBuilder(
        mysqlExe.getAbsolutePath(),
        "--defaults-file=" + myIni.getAbsolutePath()
    );
    pb.directory(new File("E:\\SQL"));
    pb.start();
    Thread.sleep(4000);
    connected = tryConnect();
}
```

改成：

```java
// 内置 MySQL 子目录（相对当前工作目录，即包根目录）
File mysqlDir = new File("mysql");
File mysqlExe = new File(mysqlDir, "bin/mysqld.exe");
File myIni    = new File(mysqlDir, "my.ini");
if (mysqlExe.exists() && myIni.exists()) {
    ProcessBuilder pb = new ProcessBuilder(
        mysqlExe.getAbsolutePath(),
        "--defaults-file=" + myIni.getAbsolutePath()
    );
    pb.directory(mysqlDir.getAbsoluteFile());
    pb.redirectErrorStream(true);
    pb.start();
    Thread.sleep(4000); // 等待 MySQL 启动（原值，勿改）
    connected = tryConnect();
}
```

**只改这一段**，其余包括上面的 `tryConnect()` 和下面的失败 `System.exit(1)` 全部保留。

### start_mysql.bat（动态生成 my.ini + 首次初始化）

要点：① 用 powershell 探测 3306 端口，已开则复用跳过；② 动态生成 `mysql\my.ini`（basedir/datadir 用本机绝对路径）；③ 若 `mysql\data\mysql` 不存在则 `mysqld --initialize-insecure`；④ 后台启动 mysqld 并轮询端口直到就绪（最多 30 秒）。my.ini 里加 `skip-grant-tables` 让 root 免密（匹配代码密码候选 `""`）。完整模板见《打包计划书.md》第 5.1 节，照抄即可。

### run.bat（用户入口）

```bat
@echo off
chcp 65001 >nul
cd /d "%~dp0"
echo 原神文字战斗模拟系统 - 启动中
call start_mysql.bat
if %errorlevel% neq 0 ( echo [错误] MySQL 启动失败 & pause & exit /b 1 )
set "CP=bin;lib\mysql-connector-j-8.0.33.jar"
java -cp "%CP%" com.genshin.rpg.Main
call stop_mysql.bat
pause
```

### stop_mysql.bat

启动 mysqld 时记录 PID 到 `mysql\mysqld.pid`，关闭时按 PID `taskkill`。若实现复杂可简化为"程序退出后保留 MySQL 运行，下次启动更快"，并在 README 说明。优先实现 PID 方式。

### 工作步骤（按序）

1. 建 `dist\genshinrpg\` 全部子目录。
2. 复制并瘦身 MySQL（按清单删）。
3. 复制 `refactored\src\` → `GenshinRPG\src\`；jar → `lib\`；sql → `db\`。
4. 改 DBConnector.java（只改路径段）。
5. 在 `GenshinRPG\` 下编译：`javac -encoding UTF-8 -d bin @sources.txt`（先 `dir /s /b src\*.java > sources.txt`），确认 0 error。
6. 写 4 个脚本 + README + .gitignore。
7. **本地验证（必做，这是验收关键）**：
   a. 停掉本机已有 mysqld（任务管理器结束 mysqld.exe 或 `net stop mysql`），模拟干净环境。
   b. **删 `GenshinRPG\mysql\data` 整个目录**，模拟全新解压。
   c. 双击 `run.bat`：应看到端口探测 → 首次 `--initialize-insecure` 初始化 → 程序启动到登录窗。
   d. 试注册一个新号、登录、进主菜单、打一关、注销，确认数据存档正常。
   e. 不关 MySQL，再次双击 run.bat：应显示"端口已开放，跳过启动"，直接进登录窗。
8. 打 zip：`GenshinRPG\` → `dist\GenshinRPG.zip`。
9. 写 `dist\PACK_NOTE.md`：瘦身明细（删了什么、留了什么、各步体积）、首次初始化验证记录、复用启动验证记录、最终 zip 体积。
