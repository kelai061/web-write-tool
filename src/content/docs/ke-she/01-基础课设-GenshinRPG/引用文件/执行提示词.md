- **输出/副作用**：返回值含义；是否改字段/写库/切页/刷新 UI。
  按钮事件处用 `// 点击后：① 调 xxx ② 改 yyy ③ 刷新 zzz` 形式就地注释。

### run.bat（写在 `refactored\run.bat`，Windows cmd 语法）

```bat
@echo off
cd /d "%~dp0"
echo [1/2] Compiling...
if not exist bin mkdir bin
dir /s /b src\*.java > sources.txt
javac -encoding UTF-8 -d bin @sources.txt
if %errorlevel% neq 0 ( echo [ERROR] Compile failed & pause & exit /b 1 )
echo [2/2] Running...
set "CP=bin;mysql-connector-j-8.0.33.jar"
java -cp "%CP%" com.genshin.rpg.Main
pause
```
