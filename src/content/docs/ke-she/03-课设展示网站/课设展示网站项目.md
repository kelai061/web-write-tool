## 项目介绍

### 这个项目是什么？
这是本站（课程设计作品集与技术实践展示平台）的设计与上线归档项目。它作为一个在线档案馆，集中展示和复盘利用 AI 辅助完成的各个课程设计项目。

### 做成了什么？
- **Markdown 自动化排版编译**：基于 Astro 5 静态站点引擎与 Starlight 文档套件，打通了本地纯文本 Markdown 向现代化、自适应、具备全文检索（Pagefind）网页的自动编译渲染流。
- **CI/CD 托管与边缘分发**：完成了 Git 版本初始化与 GitHub 代码库对接；配置了 Cloudflare Pages 构建策略，实现代码提交后自动触发全球 CDN 节点的热重载与秒级发布，并绑定了自定义域名。

### 值得看什么？
- **静态站点生成（SSG）最佳实践**：记录了从 MarkText 等本地编辑器到 Astro 框架的完整调研和配置流程。
- **自动化上线发布路径**：整理了在 Cloudflare Pages 上部署 Astro 项目时遇到的问题（如打包预设、自定义 CSS 注入等）及其实际落地方案。
